# Reconstruccion de imagenes

In this project I implemented the following image reconstruction methods: (1) filtered backprojection, (2) ART, (3) SIRT and (4) SART. The implementations are not the most efficient, my goal was to understand them and follow to the letter the algorithms described in the book: Principles of Computerized Tomographic Imaging. 

In addition, I implemented a very simple ConvNet for tumor detection.
